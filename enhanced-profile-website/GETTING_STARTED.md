# Getting Started with Enhanced Profile Website

## Quick Start Guide

### Step 1: Install Node.js

If you don't have Node.js installed:
1. Visit https://nodejs.org/
2. Download the LTS version
3. Install with default settings
4. Restart your terminal/command prompt

### Step 2: Navigate to Project

```bash
cd enhanced-profile-website
```

### Step 3: Install Dependencies

```bash
npm install
```

This will install all required packages (React, TypeScript, Tailwind CSS, etc.)

### Step 4: Start Development Server

```bash
npm run dev
```

### Step 5: Open in Browser

The website will be available at:
- **Local**: http://localhost:5173
- **Network**: Check the terminal for the network URL

## Available Commands

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build
- `npm run lint` - Run ESLint

## Troubleshooting

### Port Already in Use

If port 5173 is already in use, Vite will automatically use the next available port. Check the terminal output for the actual URL.

### Node.js Not Found

Make sure Node.js is installed and added to your PATH. Restart your terminal after installation.

### Dependencies Installation Issues

Try:
```bash
npm cache clean --force
npm install
```
